� Installation Instructions

Step 1: Install

  1. Double-click the downloaded DMG file
  2. Drag "Light Engine.app" to the Applications folder icon
  3. If you see a permission error, open Terminal and run:
    sudo chown -R $(whoami):staff "/Applications/Light Engine.app"
  4. Eject the DMG

Step 2: Security Bypass

This app is not signed with an Apple Developer certificate. Choose one method:

Option A: Right-Click (Easiest)

  1. Right-click "Light Engine" in Applications
  2. Select "Open"
  3. Click "Open" in the security dialog

Option B: Terminal Command

sudo xattr -cr "/Applications/Light Engine.app"
