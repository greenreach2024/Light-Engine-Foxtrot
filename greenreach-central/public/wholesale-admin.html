Onboarding Instructions

  1. Add Farm to Network: Use the "Farm Management" tab to add the farm to the network registry if not already added.
  2. Initiate OAuth: Click "Connect Square" button for the farm.
  3. Farm Authorization: Farm admin must click the generated OAuth link and authorize GreenReach in their Square account.
  4. Verify Connection: After authorization, the farm's status will update to "Connected".
  5. Test Payment: Create a test order to verify commission splits work correctly.
Required Square OAuth Scopes:
  • PAYMENTS_WRITE - Process payments
  • PAYMENTS_WRITE_ADDITIONAL_RECIPIENTS - Enable commission splits
  • MERCHANT_PROFILE_READ - Access merchant details
