API Documentation

Integrate GreenReach Wholesale into your custom systems using our comprehensive REST API. Authentication uses OAuth 2.0 for secure access.

Core Endpoints

GET /api/wholesale/catalog
Get available products and pricing
POST /api/wholesale/checkout/execute
Execute wholesale checkout (allocate, pay, create order)
GET /api/wholesale/orders/{order_id}
Get order details and status
POST /api/wholesale/product-requests/create
Request products not in current catalog
GET /api/wholesale/buyer-analytics
Get spending and sourcing analytics
