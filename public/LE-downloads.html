Installation Instructions

  1. Download the appropriate version for your computer
  2. macOS: Open the .dmg file and drag Light Engine to your Applications folder
  3. macOS Security: If you see a security warning, go to System Preferences → Security & Privacy and click "Open Anyway"
  4. Login: Use your same farm credentials to login to the desktop app
  5. Sync: Your farm data will automatically sync from the cloud
