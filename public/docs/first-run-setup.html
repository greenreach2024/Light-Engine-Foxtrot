First Run Setup

Touchscreen-optimized wizard for provisioning new farm servers in 6 simple steps

10 MIN
What You'll Accomplish

This guide walks you through the complete first-run setup process from unboxing to operational status. The touchscreen-optimized wizard makes provisioning fast and error-free.

Pre-Installation Checklist

Before starting the setup wizard, ensure you have:

  • Registration code from GreenReach Central (8 characters)
  • Network credentials (WiFi SSID/password if using wireless)
  • Farm details (name, address, contact information)
  • Hardware inventory list
  • Internet connectivity test completed

Hardware Requirements

  • Symcod W101M 10.1" industrial touchscreen computer (Intel N97 @ 3.6GHz, 8GB RAM, 240GB NVMe)
  • Power Supply: 12V DC adapter (included with Symcod)
  • Network: Ethernet cable (recommended) or WiFi

Step 1: Welcome Screen

The wizard starts with a welcome screen providing an overview of the 6-step process and estimated completion time (5-10 minutes).

Setup Duration

Total time: 5-10 minutes depending on network speed and number of devices.

Step 2: Network Configuration

Option A: Ethernet (Recommended)

Advantages:

  • More reliable connection
  • Higher bandwidth and lower latency
  • No credential management
  • Automatic DHCP configuration

Steps:

  1. Connect Ethernet cable to device
  2. Tap Ethernet button in wizard
  3. System automatically detects connection
  4. IP address assigned via DHCP
  5. Connection test runs automatically
Troubleshooting Ethernet
  • No Connection: Check cable is securely connected
  • No IP Address: Verify DHCP enabled on router
  • Test Failed: Check firewall settings

Option B: WiFi

When to Use: Ethernet not available, temporary setup, or mobile installations

Steps:

  1. Tap WiFi button in wizard
  2. Wait for network scan (10-30 seconds)
  3. Select your network from list
  4. Use on-screen keyboard to enter password
  5. Tap Connect
  6. Wait for connection test (10-15 seconds)

WiFi Requirements:

  • 2.4GHz or 5GHz network
  • WPA2 or WPA3 security
  • Minimum signal strength: -70 dBm
  • Outbound ports 443, 3000, 3001 open

Step 3: Farm Registration

Register your farm server with GreenReach Central to enable data synchronization and remote management.

Obtain Registration Code

  1. Log into GreenReach Central dashboard
  2. Navigate to Farms → Add New Farm
  3. Click Generate Registration Code
  4. Copy the 8-character code (e.g., A7F2B1C9)

Enter Code in Wizard

  1. Tap the registration code input field
  2. On-screen keyboard appears automatically
  3. Type the 8-character code (case-sensitive)
  4. Code is validated in real-time
Security Notes
  • Registration codes expire after 24 hours
  • Each code can only be used once
  • API credentials are encrypted at rest
  • TLS 1.3 used for all communication

Registration Verification

The wizard will:

  • Call GreenReach Central API
  • Validate registration code
  • Retrieve farm configuration
  • Download API credentials
  • Establish secure connection

Step 4: Farm Certifications

Help wholesale buyers and customers find your farm by sharing your certifications and growing practices. This information appears in the GreenReach wholesale marketplace and farm profile.

Certifications

Select any certifications your farm holds:

  • GAP Certified: Good Agricultural Practices
  • USDA Organic: Certified organic production
  • Food Safety Certified: GFSI, SQF, or other food safety standards
  • Greenhouse Grown: Certified greenhouse operation

Growing Practices

Select practices that apply to your farm:

  • Pesticide Free: No pesticides used
  • Non-GMO: No genetically modified crops
  • Hydroponic Growing: Soilless cultivation system
  • Local: Farm within 100 miles of buyers
  • Year-Round Production: Continuous crop cycles

Special Attributes

Highlight what makes your farm unique:

  • Woman-Owned Business
  • Veteran-Owned Business
  • Minority-Owned Business
  • Family Farm
  • Sustainable Practices
Why This Matters

Buyers often search for specific certifications or practices. Selecting accurate tags helps your farm appear in relevant searches and builds trust with wholesale customers.

Step 5: Hardware Detection

Automatic discovery and enumeration of connected devices. The wizard scans for:

USB Devices

  • Temperature/humidity sensors
  • CO2 sensors
  • Webcams
  • Serial adapters (FTDI, CH340)

Serial Ports (RS-485/Modbus)

  • Environmental controllers
  • Industrial sensors
  • HVAC equipment
  • Lighting controllers

Network Devices

  • IP cameras (RTSP/ONVIF)
  • Network sensors
  • PoE devices
  • IoT gateways
Scan Duration
  • USB scan: 2-5 seconds
  • Serial scan: 5-10 seconds
  • Network scan: 10-30 seconds
  • Total time: ~30-45 seconds

Device Categorization

Devices are automatically categorized:

  • Lights: LED controllers, grow lights, spectrum controllers
  • Fans: Exhaust fans, circulation fans, HVAC blowers
  • Sensors: Temperature, humidity, CO2, pH, EC, lux
  • Cameras: IP cameras, USB webcams, monitoring systems
  • Other: Uncategorized or unknown devices

Manual Override: Tap any device to view details, change type, assign to zone, or set custom name.

Step 6: Completion

Setup is complete! The wizard displays a summary of configured settings.

Final Steps:

  1. Review configuration summary
  2. Tap Go to Dashboard to access main interface

What Happens Next:

  • Farm server registers with Central API
  • Initial data sync begins
  • Hardware devices added to farm database
  • Monitoring starts immediately

Post-Setup Configuration

Farm Profile

  1. Basic Information: Farm name, legal entity, address, contact info, business hours
  2. Layout Configuration: Number of rooms, zones per room, tray capacity, growing method
  3. Crop Selection: Primary crops, growing cycles, target harvest dates, inventory SKUs

Zone Assignment

Map hardware devices to physical zones:

  1. Navigate to Setup → Zones
  2. Select a zone (e.g., "Room 1 - Zone A")
  3. Assign detected devices (lights, fans, sensors)
  4. Configure zone-specific settings

Automation Rules

Configure basic automation:

  • Climate Control: Temperature range (65-75°F), humidity (60-70%), CO2 supplementation
  • Lighting Schedule: On/off times, spectrum profiles, DLI targets (12-18 mol/m²/day)
  • Fan Control: Circulation schedule, exhaust triggers, fresh air exchange rates

Troubleshooting

Wizard Won't Start

Symptoms: Blank screen or frozen at welcome screen

Solutions:

  1. Check power supply is connected
  2. Verify device is powered on (LED indicator)
  3. Restart the device
  4. Contact support if issue persists

Network Connection Failed

Ethernet Issues:

  • Verify cable is CAT5e or better
  • Check router DHCP pool not exhausted
  • Try different port on router
  • Check for PoE compatibility

WiFi Issues:

  • Verify password is correct (case-sensitive)
  • Check signal strength (minimum -70 dBm)
  • Try 2.4GHz network if 5GHz has issues
  • Ensure WiFi not on isolated VLAN

Registration Failed

Solutions:

  1. Verify code is exactly 8 characters
  2. Check code hasn't expired (24hr limit)
  3. Verify internet connectivity
  4. Check firewall allows outbound port 443
  5. Verify Central API is online (status.greenreach.com)

No Devices Detected

USB Devices:

  • Check USB cables are securely connected
  • Try different USB port
  • Verify device is powered on
  • Run hardware detection again

Serial Devices:

  • Verify baud rate matches device (9600/19200/115200)
  • Check RS-485 termination resistors
  • Verify Modbus address configuration
  • Check wiring connections

Network Devices:

  • Verify devices on same network
  • Check devices not behind firewall
  • Verify device IP address is correct
  • Check ONVIF/RTSP enabled for cameras

Touchscreen Calibration

If touchscreen is misaligned or unresponsive:

  1. Navigate to Settings → Touchscreen → Calibrate
  2. Tap each target (top-left, top-right, bottom-right, bottom-left, center)
  3. Test calibration with test points
  4. Save calibration if accurate
  5. Restart if issues persist
