# 🤖 Farm Assistant - Quick Guide for Kids

## 💡 How to Use It

Just type what you want! The robot understands! 🤖

---

## 🌱 What Can I Ask?

### 1. **What's Ready to Pick?**
Type: **harvest** or **ready**

Examples:
- `harvest`
- `today's harvest`
- `ready`
- `pick`

### 2. **How Hot Is It?**
Type: **temp** or **hot**

Examples:
- `temp`
- `hot`
- `temperature`
- `how hot`

### 3. **Where Is My Plant?**
Type: **the plant name**

Examples:
- `basil`
- `lettuce`
- `where basil`
- `find lettuce`

### 4. **Make Lights Blink!**
Type: **blink** + **plant name**

Examples:
- `blink basil`
- `blink lettuce`
- `flash lights`

### 5. **Go to Different Pages**
Type: **page name**

Examples:
- `schedule` → See planting calendar 📅
- `inventory` → See all plants 🥬
- `heatmap` → See temperature map 🗺️

---

## 🎯 Try These!

Copy and paste these into the robot:

1. `harvest` - See what's ready today
2. `temp` - Check temperature
3. `basil` - Find basil plants
4. `blink basil` - Make basil lights blink
5. `schedule` - See planting calendar

---

## 🌟 Cool Tips

✅ **No perfect spelling needed!** - The robot is smart
✅ **Short words work!** - Just say "temp" not "temperature"
✅ **Any order works!** - "harvest today" = "today's harvest"
✅ **Can't break anything!** - Try different things!
